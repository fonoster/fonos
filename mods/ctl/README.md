# Usage
<!-- usage -->
```sh-session
$ npm install -g @fonos/ctl
$ fonos COMMAND
running command...
$ fonos (-v|--version|version)
@fonos/ctl/0.0.50 darwin-x64 node-v12.18.3
$ fonos --help [COMMAND]
USAGE
  $ fonos COMMAND
...
```
